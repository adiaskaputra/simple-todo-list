## Requirements

- [nodejs](https://nodejs.org/en/) >= v12 (LTS).
- [sequelize-cli](https://www.npmjs.com/package/sequelize-cli)
- [mysql](https://dev.mysql.com/downloads/mysql/)

## How To

Install the dependencies and devDependencies and start the server.

```sh
$ cd api
$ yarn install
$ yarn start
```
